💻 BSCS Experience Hub (Streamlit Multi-Page App)<br>
Welcome to the BSCS Experience Hub, an interactive multi-page web application built using Streamlit. This app showcases the Bachelor of Science in Computer Science (BSCS) program through engaging content, mini tools, and interactive features.

🚀 Features<br><br>
🏠 Home Page<br>
•	📊 Random Fun Fact Generator<br>
•	⏰ Real-Time Clock Display<br>
•	📈 Industry demand progress indicator<br>
•	🎯 Overview of the BSCS program<br><br>
📘 About Page<br>
•	📑 Tab-based navigation:<br>
o	Overview of BSCS<br>
o	Skills visualization using progress bars<br>
o	Career path selector<br>
•	🎓 Highlights essential competencies in Computer Science<br><br>
🧪 Projects Page<br>
•	🎮 Random Project Idea Generator<br>
•	📊 Interactive Data Visualizer (Line Chart)<br>
•	🧠 Randomized Coding Challenge Quiz<br><br>
📩 Contact Page<br>
•	📝 Functional Contact Form<br>
•	💾 Saves messages to CSV (mini database)<br>
•	🔐 Admin view for submitted messages<br>
•	📍 Map visualization<br>
•	🌐 Social media links<br>
•	⭐ Feedback rating system<br>
